# Bonus Assignment 36: Line-Based Protocol Sketch

Before writing more socket code, design a tiny line-based protocol for a networked program.

## Requirements

Define:

- at least three commands
- one success response
- one error response
- how a message ends

## Example Project Ideas

- chat server
- task service
- note sync service

## Stretch Goals

- include a login-style command
- add one command with arguments
- describe what invalid input should return
