# Mini Project 05: Contact List Formatter

Build a small terminal program that stores a few contact names in arrays and formats them cleanly.

## Goals

- practice arrays
- practice strings
- practice input handling
- practice iteration and formatting

## Features

- add up to a fixed number of contacts
- print the list in numbered format
- search for a contact by name
- count how many contacts are currently stored

## Stretch Goals

- sort the contacts alphabetically
- save the contacts to a text file later in Chapter 9
- split first and last names into separate arrays or structures
