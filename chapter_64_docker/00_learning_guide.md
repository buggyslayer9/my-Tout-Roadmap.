# Chapter 64 Learning Guide: Docker

## Learning Goals
- Understand the role of Docker in a solo-builder stack.
- Practice the core workflows of Docker.
- Connect Docker to the chapters before and after it.

## Self-Check
- Can I explain why a solo builder would choose Docker?
- Can I complete the assignment with only limited help?
- Can I see how this chapter supports a larger product build?

## Build After This Chapter
- Containerize a small app and run it locally with one service dependency.
- For a larger milestone, continue to big_project_64.md.

## Study Advice
- Read the lessons in order.
- Recreate one example by hand.
- Write down one common failure case and how you would handle it.
