# Bonus Assignment 11: Window With Two Actions

Build a GTK4 window with:

- a title label
- one button that changes the label text
- one button that resets the label text

## Requirements

- use `GtkApplication`
- connect both buttons to callbacks
- keep the UI simple and readable

## Stretch Goals

- add spacing and margins
- disable one button depending on the current state
- print a message to the terminal when the window opens
