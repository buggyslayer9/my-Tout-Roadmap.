# Mini Project 11: First Desktop Utility

Build a very small desktop utility window.

## Project Idea

A “Daily Focus” app that shows:

- a heading
- one motivational message
- one button to change the message

## Learning Goals

- create a real window
- connect callbacks
- practice widget creation and layout
- build confidence before larger GTK chapters

## Stretch Goals

- add a second label
- randomize from a list of messages
- move the messages into a helper function
