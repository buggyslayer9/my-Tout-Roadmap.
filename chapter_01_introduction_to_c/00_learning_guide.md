# Chapter 01 Learning Guide: Introduction To C

## Learning Goals
- Understand the core ideas in Introduction To C.
- Practice the lesson examples and assignment for Introduction To C.
- Connect Introduction To C to the earlier chapters in the roadmap.

## Self-Check
- Can I explain the main ideas in Introduction To C without looking at the notes?
- Can I complete the assignment with only limited help?
- Can I describe how Introduction To C will help in later projects?

## Build After This Chapter
- Build a small project that proves you can apply Introduction To C without copying the lesson examples.
- If you want a bigger milestone, see [PROJECTS.md](../PROJECTS.md).

## Study Advice
- Read the lesson files in order.
- Re-type at least one example by hand.
- Complete the assignment before moving on.
- Write down one thing that confused you and one thing that clicked.
