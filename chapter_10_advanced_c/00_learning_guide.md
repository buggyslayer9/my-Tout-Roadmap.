# Chapter 10 Learning Guide: Advanced C

## Learning Goals
- Understand the core ideas in Advanced C.
- Practice the lesson examples and assignment for Advanced C.
- Connect Advanced C to the earlier chapters in the roadmap.

## Self-Check
- Can I explain the main ideas in Advanced C without looking at the notes?
- Can I complete the assignment with only limited help?
- Can I describe how Advanced C will help in later projects?

## Build After This Chapter
- Build a simple C static library with headers, tests, examples, and a Makefile.
- If you want a bigger milestone, see [PROJECTS.md](../PROJECTS.md).

## Study Advice
- Read the lesson files in order.
- Re-type at least one example by hand.
- Complete the assignment before moving on.
- Write down one thing that confused you and one thing that clicked.
