## 🗺️ Recommended Learning Path

```
Month 1-2:  Phase 1 (C)          → K&R + Learn-C.org + Exercises
Month 3-4:  Phase 2 (GTK4)       → ToshioCP tutorial + build apps
Month 5-6:  Phase 3 (Rust)       → The Book + Rustlings + 100 Exercises
Month 7-8:  Phase 4 (Linux)      → LFS + Kernel Modules + Beej's guides
Month 9:    Phase 5 (COSMIC)     → Wayland Book + Iced + libcosmic
Month 10:   Phase 6 (Web)        → freeCodeCamp + React + Axum
Month 11:   Phase 7 (Mobile)     → Flutter docs + Angela Yu course
Month 12:   Phase 8 (DevOps)     → Pro Git + GitHub Actions + First PR
```

> **Pro Tip**: Don't just read — **build projects** after every chapter. The assignments are your minimum; go beyond them!
