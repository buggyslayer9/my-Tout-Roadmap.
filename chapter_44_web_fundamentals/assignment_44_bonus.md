# Bonus Assignment 44: Inspect and Improve a Web Page

Build a simple HTML/CSS page, then inspect it using browser DevTools.

## Requirements

- create a page with a heading, paragraph, and button
- style it with your own CSS
- inspect one element in DevTools
- change one CSS value live in DevTools
- write down what changed

## Stretch Goals

- add a console log in a small script
- inspect the console output
- identify one improvement you want to make next
