# Bonus Assignment 30: File Checker Utility

Build a small C utility that checks whether a file exists and reports success or failure cleanly.

## Requirements

- accept a file path from the user or as a simple hardcoded value
- attempt to open the file
- print success to standard output if it exists
- print an error to standard error if it does not
- return `0` on success and non-zero on failure

## Stretch Goals

- check multiple files
- report how many succeeded and failed
- turn it into a small reusable helper function
