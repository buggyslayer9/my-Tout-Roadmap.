# Chapter 29 Learning Guide: Rust Practical Projects

## Learning Goals
- Understand the core ideas in Rust Practical Projects.
- Practice the lesson examples and assignment for Rust Practical Projects.
- Connect Rust Practical Projects to the earlier chapters in the roadmap.

## Self-Check
- Can I explain the main ideas in Rust Practical Projects without looking at the notes?
- Can I complete the assignment with only limited help?
- Can I describe how Rust Practical Projects will help in later projects?

## Build After This Chapter
- Build a practical Rust project such as a CLI tool or a small terminal game.
- If you want a bigger milestone, see [PROJECTS.md](../PROJECTS.md).

## Study Advice
- Read the lesson files in order.
- Re-type at least one example by hand.
- Complete the assignment before moving on.
- Write down one thing that confused you and one thing that clicked.
