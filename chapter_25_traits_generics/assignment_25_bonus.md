# Bonus Assignment 25: Generic Display Helper

Write a Rust function that accepts any value implementing `Display` and prints it in a decorated format.

## Requirements

- use a generic type
- add a trait bound
- print the value with some surrounding text

## Example Idea

```text
=== value ===
42
============
```

## Stretch Goals

- create a second function using a different trait bound
- compare a generic approach with a non-generic one
