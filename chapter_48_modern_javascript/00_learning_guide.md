# Chapter 48 Learning Guide: Modern Javascript

## Learning Goals
- Understand the core ideas in Modern Javascript.
- Practice the lesson examples and assignment for Modern Javascript.
- Connect Modern Javascript to the earlier chapters in the roadmap.

## Self-Check
- Can I explain the main ideas in Modern Javascript without looking at the notes?
- Can I complete the assignment with only limited help?
- Can I describe how Modern Javascript will help in later projects?

## Build After This Chapter
- Build a small browser game such as snake, memory match, or tic-tac-toe.
- If you want a bigger milestone, see [PROJECTS.md](../PROJECTS.md).

## Study Advice
- Read the lesson files in order.
- Re-type at least one example by hand.
- Complete the assignment before moving on.
- Write down one thing that confused you and one thing that clicked.
